# strtoupper
strtoupper -- Makes a string uppercase.

strtoupper makes a string uppercase.

## Usage
```sh
// 1. Function form.
void func::strtoupper(string* variable)
// 2. Subroutine form.
void sub::strtoupper(string input) > output
// 3. Stream form.
void stream::strtoupper() < input > output
```
